Modals are full-featured overlay dialogs that can contain complex content. They support multiple sizes and include header, body, and footer sections.

<Primary />

<Controls />

## Sizes

- **sm**: Small (max-width: 24rem)
- **md**: Medium (max-width: 28rem, default)
- **lg**: Large (max-width: 32rem)
- **xl**: Extra large (max-width: 36rem)
- **full**: Full width with margins

## Structure

Modal uses a compound component pattern. You can access sub-components via `Modal.Header`, `Modal.Title`, etc.

### Compound Component Pattern (Recommended)

```tsx
import { Modal } from 'mario';

<Modal open={isOpen} onClose={() => setIsOpen(false)}>
  <Modal.Header>
    <Modal.Title>Modal Title</Modal.Title>
  </Modal.Header>
  <Modal.Body>
    <p>Modal content goes here</p>
  </Modal.Body>
  <Modal.Footer>
    <Button onClick={() => setIsOpen(false)}>Close</Button>
  </Modal.Footer>
</Modal>
```

### Individual Components (Backward Compatible)

You can also import and use components individually:

```tsx
import { Modal, ModalHeader, ModalTitle, ModalBody, ModalFooter } from 'mario';

<Modal open={isOpen} onClose={() => setIsOpen(false)}>
  <ModalHeader>
    <ModalTitle>Modal Title</ModalTitle>
  </ModalHeader>
  <ModalBody>
    <p>Modal content goes here</p>
  </ModalBody>
  <ModalFooter>
    <Button onClick={() => setIsOpen(false)}>Close</Button>
  </ModalFooter>
</Modal>
```

### Component Parts

- **Modal.Header**: Top section with title and optional close button
- **Modal.Title**: Main heading (automatically linked via ARIA)
- **Modal.Body**: Main content area
- **Modal.Footer**: Bottom section for actions

## Behavior

- Closes on Escape key press
- Closes when clicking backdrop
- Prevents body scrolling when open
- Focus management included
