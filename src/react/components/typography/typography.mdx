Typography component provides consistent text styling across the application. It supports semantic HTML elements and various text styles.

<Primary />

<Controls />

## Variants

- **h1-h6**: Heading levels with appropriate sizing
- **p**: Paragraph text
- **lead**: Emphasized introductory text
- **large**: Large body text
- **small**: Small text
- **muted**: Muted secondary text
- **blockquote**: Quoted text
- **code**: Inline code
- **pre**: Code block

## Custom Elements

Use the `as` prop to override the default HTML element while maintaining the variant styling.
