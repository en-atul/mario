Tooltips display additional information when users hover over or focus on an element. They appear near the trigger element.

<Primary />

<Controls />

## Positioning

Tooltips can be positioned on four sides:
- **top**: Above the element (default)
- **bottom**: Below the element
- **left**: To the left of the element
- **right**: To the right of the element

## Usage

Wrap any element with the Tooltip component. The tooltip appears on hover and disappears when the cursor moves away.

## Accessibility

Tooltips are automatically announced by screen readers. Keep tooltip content concise and informative.
