Toasts display brief, non-blocking notifications to users. They automatically dismiss after a set duration or can be manually closed.

<Primary />

<Controls />

## Variants

- **default**: Standard notification
- **success**: Success message with green styling
- **error**: Error message with red styling
- **warning**: Warning message with yellow styling

## Usage with Provider

Wrap your app with `ToastProvider` and use the `useToast` hook to programmatically show toasts:

```tsx
const { addToast } = useToast();
addToast({ title: 'Success', variant: 'success' });
```

## Duration

Set `duration` prop in milliseconds. Use `0` for persistent toasts that require manual dismissal.
