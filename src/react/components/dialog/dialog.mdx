Dialogs are modal overlays that require user interaction. They focus attention on a specific task or piece of information.

<Primary />

<Controls />

## Structure

Dialogs are composed of several sub-components:
- **DialogHeader**: Contains title and description
- **DialogTitle**: Main heading (required for accessibility)
- **DialogDescription**: Supporting text
- **DialogContent**: Main content area
- **DialogFooter**: Action buttons

## Behavior

- Opens/closes based on `open` prop
- Closes when clicking the backdrop
- Prevents body scrolling when open
- Includes backdrop overlay

## Accessibility

Always include a DialogTitle for screen reader support. Use DialogDescription to provide additional context.
