Skeleton loaders provide visual feedback while content is loading. They help maintain layout stability and improve perceived performance.

<Primary />

<Controls />

## Usage

Use skeletons to indicate loading states. Customize the size and shape using className to match your content layout.

## Best Practices

- Match skeleton dimensions to actual content
- Use multiple skeleton elements for complex layouts
- Replace with actual content as soon as it's available
