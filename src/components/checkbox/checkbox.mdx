Checkboxes allow users to select one or more options from a set. They work well with the Label component for accessible forms.

<Primary />

<Controls />

## Usage

Always pair checkboxes with labels using the `htmlFor` attribute. Use checkboxes when multiple selections are allowed.

## States

- **checked**: Selected state
- **disabled**: Non-interactive state
- **indeterminate**: Partially selected (use `indeterminate` prop)
