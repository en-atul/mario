<Title>Welcome to Mario Design System</Title>

A modern React component library built with TypeScript, Tailwind CSS, and Storybook.

## Getting Started

Browse the components in the sidebar to explore the design system. Each component includes:

- **Interactive examples** - See components in action
- **Documentation** - Learn how to use each component
- **Controls** - Experiment with different props and variants
- **Accessibility** - Built with a11y best practices

## Component Categories

- **Forms** - Input, Select, Textarea, Checkbox, Radio, Switch, Label
- **Actions** - Button, Link
- **Feedback** - Toast, Tooltip, Badge, Skeleton
- **Layout** - Separator, Typography
- **Overlays** - Modal, Dialog, Dropdown
- **Navigation** - Tabs

## Design Tokens

Explore the design tokens to understand the foundation of the design system:

- **Colors** - Color palette and semantic colors
- **Typography** - Font sizes, weights, and line heights
- **Spacing** - Consistent spacing scale
- **Radius** - Border radius values
- **Elevation** - Shadow and depth tokens
- **Motion** - Animation durations, easing, and presets
- **Icons** - Available icon library

## Patterns & Guidelines

Learn common UI patterns and best practices:

- **Form Layout** - Standard form structures and validation patterns
- **Error Handling** - How to display and handle errors
- **Modal Usage** - When and how to use modals effectively

## Usage

```tsx
import { Button, Input, Modal } from 'mario';

function App() {
  return (
    <div>
      <Button variant="primary">Click me</Button>
      <Input placeholder="Enter text..." />
    </div>
  );
}
```
